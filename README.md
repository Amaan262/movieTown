<h1>MovieTown</h1>
🏅Responsive React Movies App.
<br>
🏅Made with TMDB API.
<br>
🏅Two Threads available for Movies and Tv Shows
<br>
🏅Storyline, casts and trailer can be viewed on the movie page.

🏅Adaptive layout for different screen size.
